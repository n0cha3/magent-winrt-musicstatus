# SMTC Music status for magent

This plugin allows you to fetch the media playback status from WinRT's Global SMTC and display it in magent.
It works for all version from 5.x up to 6.3 (afterwards this functionality has been removed from the messenger)

![preview](example.png)

## P.S 
At this moment you must manually patch the exe file in order to use this plugin.
